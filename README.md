# README

Repositório do jogo Dom-que-shot

## Setup inicial

Versão do Node.js: 16.13.2

Clonar o repositório e executar:

```
npm i
```

## Rodando localmente

```
npm run dev
```

## Rodando os testes

```
npm run test
```

## Rodando ESlint

```
npm run lint
```

## Gerando Build

```
npm run build
```
